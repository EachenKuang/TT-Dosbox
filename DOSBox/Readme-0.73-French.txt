Ce fichier archive contient un pack de traduction en français des commandes et messages
ainsi que du fichier de configuration s'affichant sous dosbox 0.73.
Ce pack n'est pas compatible avec les versions précédentes de dosbox, consulter le site pour
télécharger la version appropriée.


Procédure d'installation facile pour Windows :


1a. Pour Windows Vista
Copier le fichier "francais.lng" et "dosbox-073.conf" contenus dans l'archive vers le répertoire de dosbox 0.73 pour votre profil
"C:\Users\[votre nom]\AppData\Local\DOSBox" par défaut
ATTENTION, faites une copie de suavegarde du fichier "dosbox-073.conf" déjà présent par sécurité.

1b. Pour Windows XP
Copier le fichier "francais.lng" et "dosbox-073.conf" contenus dans l'archive vers le répertoire de dosbox 0.73 pour votre profil
"C:\Documents and Settings\[votre nom]\Local Settings\Application Data\DOSBox" par défaut
ATTENTION, faites une copie de suavegarde du fichier "dosbox-073.conf" déjà présent par sécurité.

-------------

2. Le fichier de configuration est déjà modifié pour prendre en compte le fichier de traduction.
Ouvrir avec un éditeur de texte ou notepad le fichier "dosbox-073.conf" ou cliquer sur 
le raccourci "Edit configuration" dans la section "Configuration" du groupe de programmes "DOSBox-0.73"


3. Lancer Dosbox. Il s'affiche maintenant en français!  :)

6. A l'invite de commande si le clavier n'est pas automatiquement configuré pour utiliser un clavier français, taper:
keyb fr


nota:
- Il est possible que la traduction ne concerne pas tous les messages d'information utilisables du programme.
- N'ont été traduits que les messages et les commandes disponibles en lançant 
"CONFIG -writelang nom_de_fichier" sous DOSbox.
- Il subsite dans cette version un problème avec l'écriture du fichier de configuration à partir d'un fichier de langue en lançant
"CONFIG -writeconf nom_de_fichier" sous DOSbox.
Le fichier créé se trouve dans le répertoire d'installation de DOSbox ("C:\Program Files\DOSBox-0.73" par défaut).
Le fichier contient des sauts de ligne entre chaque commentaire traduit et certains termes ne sont pas traduits.
L'équipe de développement est au courant et résoudra peut-être cette erreur dans une future version.
-Si vous trouvez des fautes de frappe ou d'orthographe, n'hésitez pas les signaler à l'équipe de développement qui transmettra
ou bien directement sur le forum http://www.abandonware-forums.org à mon intention.
